1. Giải thích các câu lệnh:
- git init: khởi tạo một thư mục được quản lý bởi git
- git add <file_name>: thêm file_name vào vùng Staging để chuẩn bị cho commit
- git add .: thêm toàn bộ file có trên vùng working directory vào vùng Staging để chuẩn bị cho commit
- git commit -m"message": tạo 1 commit để commit các file ở vùng Staging lên repository area với nội dung "message"
