1. Giải thích các câu lệnh:
- git commit --amend: mở giao diện trình soạn thảo ViM để chỉnh sửa lại nội dung của commit message gần nhất (bấm i để insert và :wq để write and quit)
- git commit --amend -m"<message>": trực tiếp thay đổi nội dung của commit message, với nội dung mới bằng <message>
- git restore --staged <file>: đưa các file đã được add vào vùng Staging quay trở lại vùng Working
- git reset HEAD~1: revert commit, đưa 1 commit mới nhất đã được đẩy vào Repo quay trở lại vùng Working
